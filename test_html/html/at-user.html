一、什么是AT

AT是先知的区块链代币，通过以太坊以ERC20标准发行。
AT的总量为10亿，永不增发。

二、如何获取AT

用户可通过以下几种方式获得AT

  1. 文章转载数量激励：发布优质转载内容，被先知管理员审核通过后可获得AT内容激励
  2. 文章转载热度激励：根据内容发布后每日新增的点赞、阅读、评论等数据计算出内容的热度，根据该活跃度可获得AT内容活跃激励
  3. 问答发布数量激励：发布优质提问和回答内容，被先知管理员审核通过后可获得AT内容激励
  4. 问答发布热度激励：根据回答内容发布后每日新增的被赞同数、评论等数据计算出回答的热度，提问的热度与该提问下的回答热度正相关，根据提问和回答热度可获得AT问答热度激励
  5. 用户活跃激励：用户活跃度与用户当日使用先知APP的使用时长、点赞、评论、转发等行为有关，根据该活跃度可获得AT用户活跃激励
  6. 收矿工激励：用户注册完成后，可邀请好友注册，被邀好友即会成为你的矿工，你可永久获得矿工挖矿激励的20%作为收矿工激励
  7. 锁仓激励：在先知平台上锁定一定量的AT的用户，将获得相应的锁仓激励，锁仓用户能获得更多的每日内容发布数量，以及计算挖矿激励时会乘以锁仓激励倍数（例：当您今天的用户活跃度为500，通过AT锁仓获得的锁仓激励倍数为2，那么您今天的真实用户活跃度=500*2），锁仓超过5万AT的用户，还可以获得每日锁仓利息激励，锁仓数额越大，年化利率越高
  8. 注册激励：新用户注册先知APP后，将获得一定数量的AT奖励

四、激励池分配比例

先知将设定每日发放的激励代币总量上限，并根据目前产品形态共划分成5个激励池，每个激励池根据对社区生态的贡献价值会在发放代币总量中占有不同比例，每个激励池占全部发放代币总量的占比如下：

  • 文章转载数量激励池：25.2%
  • 文章转载热度激励池：4.8%
  • 问答数量激励池：21.3%
  • 问答热度激励池：32%
  • 用户活跃度激励池：16.7%

五、文章转载数量激励规则

按当日用户发帖审核通过数与锁仓激励倍数的乘积占先知平台所有用户发帖审核通过数与锁仓激励倍数的乘积之和的占比均分激励池中的AT，次日上午8点发放前一日所获得的激励，用户每日提交文章数量、提问数量、回答数量及锁仓激励倍数与在先知平台内锁定的AT数量有关。

六、文章热度激励规则

按文章发布审核通过后当日的文章热度占比均分激励池中的AT，文章热度与该内容每日新增的阅读、评论、点赞、转发以及发布者锁仓激励倍数等数据相关，次日上午8点发放该内容前一日所获得的激励。

七、问答数量激励规则

按当日用户提问数、回答数与锁仓激励倍数的乘积占先知平台所有用户发帖审核通过数与锁仓激励倍数的乘积之和的占比均分激励池中的AT，次日上午8点发放前一日所获得的激励，用户每日提交文章数量、提问数量、回答数量及锁仓激励倍数与在先知平台内锁定的AT数量有关。

八、问答热度激励规则

按提问和回答发布后当日的问答热度占比均分激励池中的AT，热度与该问答每日新增的赞同数、评论数以及发布者锁仓激励倍数等数据相关，次日上午8点发放该内容所获得的激励。

九、用户活跃激励规则

活跃度与用户当日使用先知APP的使用时长、点赞、评论、转发以及发布者锁仓激励倍数等行为有关，按用户活跃度占当日总体用户活跃度的占比均分激励池中的AT。

十、收矿工激励规则

  1. 用户注册后，可通过分享邀请链接邀请好友注册
  2. 被邀请好友注册后，即成为你的矿工
  3. 你的所有矿工每日通过使用先知平台获得的用户活跃激励，通过发布内容获得的内容发布及热门激励，你均可获得该激励总和的20%作为你的收矿工激励
  4. 收矿工数量不设上限，收矿工激励永久有效
